## Installation

```bash
pnpm add noumen
```

noumen requires Node.js 18 or later.

## Setup

Every noumen agent needs two things: an **AI provider** and a **sandbox**.

<Steps>
  <Step>
    ### Choose a provider

    noumen ships six providers. Each takes an API key and an optional default model.

    <Tabs items={["OpenAI", "Anthropic", "Gemini", "OpenRouter", "Bedrock", "Vertex"]}>
      <Tab value="OpenAI">
        ```bash
        pnpm add openai
        ```

        ```typescript
        import { OpenAIProvider } from "noumen/openai";

        const provider = new OpenAIProvider({
          apiKey: process.env.OPENAI_API_KEY!,
          model: "gpt-4o", // default
        });
        ```
      </Tab>
      <Tab value="Anthropic">
        ```bash
        pnpm add @anthropic-ai/sdk
        ```

        ```typescript
        import { AnthropicProvider } from "noumen/anthropic";

        const provider = new AnthropicProvider({
          apiKey: process.env.ANTHROPIC_API_KEY!,
          model: "claude-sonnet-4-20250514", // default
        });
        ```
      </Tab>
      <Tab value="Gemini">
        ```bash
        pnpm add @google/genai
        ```


    </Tabs>
  </Step>

  <Step>
    ### Choose a sandbox

    A sandbox bundles a filesystem and a shell into one object. All tool I/O routes through it. For local development, use the unsandboxed Node.js sandbox:

    ```typescript
    import { LocalSandbox } from "noumen";

    const sandbox = LocalSandbox({ cwd: "/path/to/your/project" });
    ```

    For production or untrusted agents, swap in a sandboxed implementation like `SpritesSandbox` (remote containers), `DockerSandbox`, or `E2BSandbox`. See [Virtual Infrastructure](/docs/virtual).
  </Step>

  <Step>
    ### Create a Code instance

    The `Code` class wires everything together:

    ```typescript
    import { Code } from "noumen";

    const code = new Code({
      aiProvider: provider,
      sandbox,
      options: {
        sessionDir: ".noumen/sessions",
        model: "gpt-4o",
        maxTokens: 8192,
        autoCompact: true,
        projectContext: true,
      },
    });
    ```
  </Step>

  <Step>
    ### Run a prompt

    Create a thread and iterate over the stream events:

    ```typescript
    const thread = code.createThread();

    for await (const event of thread.run("Fix the failing test in utils.test.ts")) {
      switch (event.type) {
        case "text_delta":
          process.stdout.write(event.text);
          break;
        case "tool_use_start":
          console.log(`\nUsing tool: ${event.toolName}`);
          break;
        case "tool_result":
          console.log(`Result: ${event.result.content.slice(0, 100)}`);
          break;
        case "message_complete":
          console.log("\n--- Done ---");
          break;
        case "turn_complete":
          console.log(`Tokens used: ${event.usage.total_tokens}`);
          break;
      }
    }
    ```
  </Step>
</Steps>

## Full example

```typescript
import { Code, LocalSandbox } from "noumen";
import { OpenAIProvider } from "noumen/openai";

const code = new Code({
  aiProvider: new OpenAIProvider({ apiKey: process.env.OPENAI_API_KEY! }),
  sandbox: LocalSandbox({ cwd: process.cwd() }),
  options: {
    sessionDir: ".noumen/sessions",
    autoCompact: true,
  },
});

const thread = code.createThread();

for await (const event of thread.run("Add input validation to the signup handler")) {
  if (event.type === "text_delta") process.stdout.write(event.text);
  if (event.type === "tool_use_start") console.log(`\n[${event.toolName}]`);
}
```
